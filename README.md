## 🧩 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/navanithaadhav/Portfolio.git
```

2. Navigate to the project folder

```bash
cd Portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
